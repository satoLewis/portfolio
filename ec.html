Experiência Criativa

Projeto Jogo: Nesse trabalho, eu e meu grupo realizamos um jogo Fully Top-Down Shooter, no qual o protagonista
deveria correr e matar seus colegas de trabalho em uma competição realizada na empresa em que trabalhava,
para que dessa forma ele chegasse no boss final, seu próprio chefe. Para a realização desse jogo, foi utilizado a
plataforma Unity para a codificação do jogo e o aplicativo Aseprite para a criação da arte do jogo.

Projeto no Processing: No segundo trabalho realizado no semestre, eu e meu grupo realizamos um layout via o
aplicativo do Processing. Nossa ideia foi a criação de um espaço onde o usuário pudesse armazenar e abrir
projetos de Word ou outra plataforma de anotações e nela pudesse editar de maneira mais organizada, além
de ainda haver opções para o usuário poder usufruir do espaço de forma eficar e tranquila.

Projeto do site: No último trabalho do semestre, nosso grupo decidiu realizar um site de assistência técnica
para periféricos tecnológicos, o HardwareFix. Neste trabalho realizamos todos os requisitos orientados pelos
professores, uma aba com imagens, uma com vídeos, uma onde o usuário pudesse registrar as informações
de cadastro e enviar para o banco de dados, e por último uma aba que puxasse os dados registrado no banco
e os colocasse numa tabela. Foi utilizado as linguagens de html, javaScript e php.
